## Features

| Category               | Details                                                                                   |
| ---------------------- | ----------------------------------------------------------------------------------------- |
| **Transactions**       | Add income and expense transactions with category, account, note, and date                |
| **Analytics**          | Daily/monthly trends, category breakdown, weekly comparison, and smart insights           |
| **Dashboard**          | Balance overview, monthly overview, quick actions, and recent transactions                |
| **Accounts**           | Multi-account support: Cash, Bank Account, UPI App, and Credit Card                       |
| **Categories**         | 16 expense + 8 income default categories, with custom category support                    |
| **Budgets**            | Per-category monthly/weekly budgets with threshold-based alerts                           |
| **Recurring**          | Set daily / weekly / monthly / yearly recurring transactions                              |
| **Smart Input Parser** | Parse natural-language text like "Add 200 for food" into transaction fields               |
| **App Lock**           | PIN lock with optional biometric unlock (Face ID / Fingerprint)                           |
| **Cloud Backup/Sync**  | Optional Supabase auth + push/pull sync across devices while keeping local-first behavior |
| **Offline First**      | All data works locally via MMKV; cloud sync is optional                                   |
| **INR Native**         | ₹ currency throughout by default                                                          |
