PFT - Expense Tracker

A complete personal finance tracker built for offline-first usage, with rich analytics, secure app lock, recurring expenses, budget alerts, and optional cloud sync.

Download on Indus Appstore

Why PFT

  • Offline-first by design with fast local storage.
  • Track expenses and income across multiple accounts.
  • Set budgets and receive threshold-based spending alerts.
  • Use recurring transactions for rent and subscriptions.
  • Secure with PIN plus optional biometric unlock.
  • Optional Supabase backup and sync across devices.

All Features in Detail

PFT includes everything needed for day-to-day money management, from simple transaction capture to advanced spending insights.

Transactions

Add income or expense entries with account, category, note, and date. Edit existing entries anytime.

Analytics

Visualize daily and monthly trends, category spend split, and week-over-week movement with smart insights.

Dashboard

See current balance, monthly overview, quick shortcuts, and recent transactions on one home screen.

Accounts

Manage Cash, Bank Account, UPI App, and Credit Card balances separately for accurate tracking.

Categories

Start with built-in income and expense categories, then add your own custom categories and icons.

Budgets

Create weekly or monthly category budgets and get alerts when spending reaches your threshold.

Recurring Transactions

Automate repeating expenses with daily, weekly, monthly, or yearly recurrence settings.

Smart Input Parser

Use natural-language text like “Add 200 for food” to quickly prefill transaction details.

App Lock

Protect your data using a PIN, with optional biometric authentication for faster secure access.

Cloud Backup and Sync

Sign in and sync with Supabase to keep your data backed up and available across multiple devices.

Offline First

All core features work without internet using MMKV storage. Cloud is optional, not required.

INR Native

Built around Indian users with ₹ defaults and finance flows tailored to local usage patterns.

Built with Expo + React Native, optimized for smooth use on Android and iOS.
